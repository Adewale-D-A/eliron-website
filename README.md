# ELIRON website

> Eliron is a consulting firm dedicated to establishing credibility and clarity

### <b>Project Technical Prerequisite</b>:

> TailwindCSS is used for the CSS styling, therefore, technical knowledge of tailwindCSS frameworks, Next.JS and TypeScript are prerequisite to collaborate on this project.
